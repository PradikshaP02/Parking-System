Description of the project
Parking is one of the major problems faced by all the vehicle owners mainly in cities where vehicle traffic is really huge.
Our project mainly focuses on the concept of organized Parking system.
Talking about the features of this Simple system,
 
•	The user can add Vehicle details (Vehicle Number) in the program and it’s not time-consuming. 

•	This System makes it easy to store records of each (Arrival & Departures of vehicles). This mini project is easy to operate and understand by the users. 

•	This simple mini project in C will also use Graphic features to display the Parking area.

•	This project is helpful for managing parking information by adding, updating, removing, viewing and searching for details. 

•	Our project calculates the duration of the parking and charges accordingly.

•	The receipt is also generated while the car leaves.
